## Boolean 值是？

- 区分大小写
- 不同于数值
  - true 不等于 1， false 不等于 0

## Boolean 对象是？

- 对应布尔值的引用类型
- 创建一个 Boolean 对象？
  - 使用 Boolean 构造函数并传入 true 或 false
  - `let booleanObject = new Boolean(true);`
