# String Split, Join-拆合方法

`let str = hello, world`
`let arr = [hello,  world]`

- 字符串拆成数组：
  - `str.split(,) // [hello, world]`
- 数组组成字符串：
  - `arr.join(,) // hello, world`
  - `arr.join(o) // helloo world`
